About Website

     This website is created using Hugo, a Static Web Generator. Created for documentation of my adventure migrating from Windows to Linux, and might even post some tips and trick on using Linux to replace Windows, and share softwares that i used. Also my experiment on coding CSS, JS, and HTML.

Question & Answer

  • Why did i make this website?

Nothing special, just want to share the experience. Also my experiment on coding CSS, JS, and HTML.

  • Why did i move to Linux?

I have a lot of problem with Windows 10, experiencing a lot of bugs. And worse, the Microsoft Support don’t tell the “good” solution to solve it, they just ask you to do 2 type of scan command if it’s dont work reinstall, it’s a bad solution in my opinion. I did reinstall it a few times, but in the end… the problem keep coming back after a few days.

The thing that i hate the most about Windows is that they force you to buy their OS when you buy Laptop from certain brand/shop.

  • Must i use Linux?

Depends, most workflow requires either Mac or Windows, because the program simply not supported on Linux.

A person from a community that don’t know much about Linux says “Dont use linux, if you don’t use Linux “restricted” tools.”. NO, you don’t have to do that. I have a friend that only use Linux just for Office Suite, also my sister use it only for Office Suite.

TLDR, there’s no special requirement to use linux, but if it’s not suit your workflow then you don’t have to use it.
